Вам дан контроллер домена lab1.local. Требуется получить имена входа всех пользователей, 
входящих в группу безопасности Administrators (в том числе не напрямую, а через принадлежность 
другим группам). 
Результаты записать в файл c:\admins.txt в алфавитном порядке

Пример файла c:\admins.txt:

Admin
XXX
