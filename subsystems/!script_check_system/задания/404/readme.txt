Вам дан контроллер домена lab1.local. Требуется найти пользователей, которые 
являются владельцем хотя бы одного файла с расширением .mp3 внутри папки 
c:\users (и вложенных подпапок) данного сервера.

Для каждого такого пользователя выведите в файл c:\musers.txt 
через пробел имя входа, имя, фамилию пользователя, количество таких файлов и их объём, 
округленный до мегабайт.
Строки файла должны быть упорядочены в алфавитном порядке.

Пример файла c:\musers.txt:
vasya Василий Иванов 3 9
petya Петр Петров 100 345
