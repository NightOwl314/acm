Установка архива задач

Требования к системе(минимальные)

  • Операционная система Windows 2000 professional/server
  • Процессор PII 400 MHz
  • Память 128 Mb
  • Место на диске 200 Mb

Установка ПО производится в следующей последовательности:

  1. web-сервер Apache 1.3 и модуль FCGI для него.
  2. интерпритатор Perl и следующие модули ppm для него:
    1. FCGI
    2. DBI
    3. DBD-InterBase
  3. СУБД InterBase 6.0 (можно FireBird)
  4. IBExpert (с его помощью работать с базой гораздо удобнее)
Для нормальной работы служба защиты программного обеспечения должна быть отключена!
