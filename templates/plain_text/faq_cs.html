Решение задач на Visual C#

Пример решения задачи A+B:
   

using System;

public class Sum
{
    private static void Main()
    {
        string[] tokens = Console.ReadLine().Split(' ');
        Console.WriteLine(int.Parse(tokens[0]) + int.Parse(tokens[1]));
    }
}
Примечание. Данное решение не является самым эффективным. Например, при посимвольном вводе программа может работать быстрее и использовать меньше памяти

Параметры командной строки компилятора:
csc /o+ /r:System.Numerics.dll name.cs
