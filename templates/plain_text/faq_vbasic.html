Решение задач на Visual Basic.NET

Пример решения задачи A+B:
   

Module module1

Sub Main()
  Dim v() As String = Console.ReadLine().Split(" ")
  dim a as integer = integer.Parse(v(0))
  dim b as integer = integer.Parse(v(1))
  dim c as integer = a + b
  Console.WriteLine(c)
End Sub

End Module
Данное решение не является самым эффективным. Например, при посимвольном вводе программа может работать быстрее и использовать меньше памяти

Параметры командной строки компилятора:
vbc /optimize /reference:System.Numerics.dll name.vb
